# 저탄소 축산물 실측 대시보드 - 기능 구현 체크리스트

## 1. 데이터 통합

### ✅ 메탄캡슐의 개체별 메탄 데이터 수집
- **구현 완료**: FloorPlanView.tsx
- 한우/젖소 농장에서 개체별 캡슐 센서 표시
- 우사별로 개체 위치 및 캡슐 센서 시각화
- 마우스 호버 시 "캡슐센서 - 장내발효 측정" 툴팁 표시
- 농장 규모별 개체 수 차등 적용 (소규모: 12개, 대규모: 18개/우사)

### ✅ 환풍구 센서의 가스 농도, 환기량, 온·습도 등 환경변수 수집
- **구현 완료**: GasGauges.tsx, dataGenerator.ts
- 실시간 가스 농도 모니터링 (CH4, CO2, N2O, NH3)
- 최근 1시간 평균 표시
- 센서별 데이터 수집 및 평균 계산
- 센서 배치: 고정 센서 (내부), 외부 센서 (기준선)
- 축종별 최적화된 센서 배치 (한우: 16개, 젖소: 13개, 돼지: 21개)

### ⚠️ AI 기반 농장별 메탄 배출량 산정
- **부분 구현**: LCABreakdown.tsx, dataGenerator.ts
- IPCC Tier 2 배출계수 기반 계산 (규칙 기반)
- AI 모델은 미구현 (현재는 배출계수 × 사육두수 방식)
- 개선 제안 생성은 규칙 기반 알고리즘
- **향후 개선 필요**: 실제 AI/ML 모델 적용

## 2. 데이터 활용 - 농가용

### ✅ 실시간 배출 현황 모니터링
- **구현 완료**:
  - **탄소배출량 현황** (LCABreakdown.tsx)
    - 일간/주간/월간 배출량 실시간 표시
    - 어제/지난주/전월 대비 비교
    - 직접 배출 (가축, 분뇨) vs 간접 배출 (전력, 연료, 기타)
    - 배출원별 구성 비율 시각화

  - **실시간 가스 농도** (GasGauges.tsx)
    - CH4, CO2, N2O, NH3 농도 실시간 표시
    - 최근 1시간 평균
    - 정상/주의/경고 기준선 표시
    - 경고 시 즉각 행동 촉구 메시지

  - **센서 배치도** (FloorPlanView.tsx)
    - 축사별 센서 위치 실시간 표시
    - 개체별 캡슐 센서 위치
    - 내부/외부 센서 구분

### ✅ 저탄소 인증 관리
- **구현 완료**:
  - **인증 등급 표시** (FarmOverview.tsx)
    - A/B/C 등급 명확한 색상 구분
    - 농장별 인증 정보 상단 표시

  - **인증 성과 메시지** (FarmOverview.tsx)
    - "지금처럼만 유지하면 저탄소 농장 유지 가능해요!" (A등급 유지)
    - "좋아요! 이대로 계속 개선하면 A등급 달성 가능합니다" (B등급 개선중)
    - "주의! 이대로 가면 인증 등급이 하락할 수 있어요" (배출량 증가)
    - "개선이 필요합니다. 인증이 사라질 수 있어요" (C등급)

  - **배출량 추이 차트** (EmissionChart.tsx)
    - 월별 배출량 추이 그래프
    - 직접/간접 배출 분리 표시

  - **지역별 인증 현황** (RegionalStatsPage.tsx)
    - 17개 시도별 인증 농장 통계
    - 축종별 분류 (한우/돼지/젖소)
    - 정부 통계 스타일 테이블

### ✅ 환기, 사료 운영 최적화
- **구현 완료**:
  - **개선 제안 시스템** (LCABreakdown.tsx)
    - 실시간 문제 감지 및 즉각 행동 제안
    - 기한별 조치 사항 (오늘 안에/이번 주 안에/이번 달 안에)

  - **환기 최적화**:
    - 전력 소모량 증가 시: "오늘 안에 사용하지 않는 조명·환기팬 전원 차단"
    - 가스 농도 경고 시: "즉시 환기 시스템 가동 및 창문 개방하세요"
    - 가스 농도 주의 시: "환기 시스템 가동 시간을 늘리세요"

  - **사료 운영 최적화**:
    - ⚠️ 사료 데이터 수집은 되지만 배출량 계산에서 제외 (인증 평가 기준 맞춤)
    - 가축 배출 증가 시: "저메탄 첨가제(3-NOP, 해조류) 급여 시작"
    - MSY 생산성 개선 제안

  - **분뇨 관리**:
    - 분뇨 배출 증가 시: "오늘 안에 피트 내 슬러리 즉시 처리"
    - 처리 주기 관리: "월 1회 이상 필수"
    - 바이오가스 포집 시설 제안

  - **연료 최적화**:
    - 연료 소모 증가 시: "오늘 안에 난방기 설정온도 1-2도 낮추기"
    - 장비 공회전 중단 제안

## 4. 미구현/개선 필요 사항

### ⚠️ AI 기반 배출량 산정
- 현재: 규칙 기반 (IPCC 배출계수)
- 필요: 머신러닝 모델 적용
- 개체별 메탄캡슐 + 환경 센서 데이터 → AI 모델 → 정밀 배출량 예측

### ⚠️ 실제 센서 연동
- 현재: 시뮬레이션 데이터
- 필요: 실제 IoT 센서 API 연동
- WebSocket/MQTT 실시간 데이터 스트리밍

### ⚠️ 환기량 데이터
- 현재: 가스 농도만 표시
- 필요: 환기 시스템 가동률, 풍량(CFM) 데이터 추가

### ⚠️ 온·습도 데이터
- 현재: 미표시
- 필요: 축사 내부 온습도 센서 데이터 추가
